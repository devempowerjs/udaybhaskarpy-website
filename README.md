# udaybhaskarpy-website  

This repository contains the source code for the Uday Bhaskar Portfolio Website, developed by Mayank Raj (devempowerjs). The project delivers a clean, responsive, and aesthetic portfolio experience with smooth animations and a modern frontend stack.  

## Overview  

The website is built to establish a strong digital presence for the client, showcasing projects, skills, and professional details in a visually appealing and structured format. It focuses on seamless user experience, performance, and modern design principles.  

## Key Features  

Fully responsive design optimized for all screen sizes  
Smooth and engaging animations for enhanced user experience  
Clean and aesthetic user interface with modern design patterns  
Component based architecture for scalability  
Optimized performance and fast loading  
Structured portfolio and project showcase  

## Tech Stack  

TypeScript  
React  
Tailwind CSS  

## Architecture  

The project follows a modular and component driven structure, ensuring reusability and maintainability. UI components are designed to be scalable, while logic and presentation are kept separate for better code clarity.  

Utility first styling is used to maintain consistency and accelerate development.  

## Performance Considerations  

Efficient rendering using React best practices  
Optimized styling with Tailwind CSS  
Minimal and structured imports to reduce bundle size  
Responsive layouts to ensure smooth performance across devices  

## Getting Started  

To run the project locally, follow these steps  

Install dependencies  
`npm install`  

Start the development server  
`npm run dev`
